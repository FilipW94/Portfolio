Description

Goal of this project was to create a map that have all the functionality you would normally find in modern games.

Features

  • Accurate world representation in terms of distance
  • Usable with gamepad
  • Zooming
  • Panning
  • Player marker
  • Placeable waypoint
  • Points of interest
  • Information about hovered elements
